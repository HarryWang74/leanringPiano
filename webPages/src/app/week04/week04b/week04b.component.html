P125 输出音质提升研究

P125 输出

用了几周，开始感觉电琴的speaker音质不足，需要将电琴内置钢琴采样输出到外置播放器，提高演奏音质

  • USB HOST
  • 1/4 inch (6.3mm)

使用电脑扬声器

Macbook 连接使用 Mac 显示器内置扬声器，音效还可以。

思路 1

  • 使用 USB host 把电琴作为 MIDI keyboard 使用
  • 电脑接收到 MIDI 信号后使用软音源或 AWD 内置音源播放
  • 没有使用 YAMAHA 内置音源

思路 2

  • 通过电琴的 1/4 inch 转换为 USB 依旧使用显示器扬声器
  • 1/4 to usb c
  • Double 1/4 to 3.5
  • 3.5mm to usb
  • 使用 YAMAHA 内置音源
